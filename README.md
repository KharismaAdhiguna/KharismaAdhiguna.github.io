# Git Basic Commands

1. Initialization
```sh
$ git init
```

2. Adding file to repository
```sh
$ git add filename
```
3. Make a commit/comment
```sh
$ git commit -m "your comment"
```
4. Pushing file into online repository
```sh
$ git push
```
